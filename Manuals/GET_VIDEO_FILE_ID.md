# Как получить file_id для видео-кружка

## Проблема
Ошибка: `Bad Request: wrong file identifier/HTTP URL specified`

Это значит что `WELCOME_VIDEO_FILE_ID` в `.env` неправильный или устарел.

## Решение

### Шаг 1: Запусти скрипт
```bash
python get_video_file_id.py
```

### Шаг 2: Отправь видео-кружок
1. Открой бота в Telegram
2. Запиши видео-кружок (круглое видео) или отправь готовое
3. Отправь его боту

### Шаг 3: Скопируй file_id
Скрипт выведет что-то вроде:
```
============================================================
✅ Видео-кружок получен!
============================================================

file_id: DQACAgIAAxkBAAIBY2Z...

Добавь эту строку в .env:
WELCOME_VIDEO_FILE_ID=DQACAgIAAxkBAAIBY2Z...

============================================================
```

### Шаг 4: Обнови .env
Добавь или обнови строку в `.env`:
```env
WELCOME_VIDEO_FILE_ID=DQACAgIAAxkBAAIBY2Z...
```

### Шаг 5: Перезапусти бота
```bash
docker-compose down
docker-compose up -d
```

## Важно!
- file_id привязан к конкретному боту
- Если сменишь бота (BOT_TOKEN), нужно получить новый file_id
- file_id может устареть через несколько месяцев — просто получи новый

## Альтернатива: без видео-кружка
Если не хочешь использовать видео-кружок, просто оставь переменную пустой:
```env
WELCOME_VIDEO_FILE_ID=
```

Бот будет работать без приветственного видео.
